<h1>Using the terminal</h1>

The terminal is a powerful tool that allows us to execute code on a machine.

STEP1:
Opening the terminal

The terminal app can be found by going to the applications folder then finding the utilities folder. A terminal window or shell will open up.

STEP2:
The shell has a prompt and a location in the file directory system of the computer you are currently using. You can find out what folder and files are in the is directory using…

<code>ls</code>

You can move around the directory structure by using 

<code>cd path_location_of_folder</code>

You can access a text editor by typing

<code>nano</code>

You can create a new directory in the current location using

<code>mkdir name_of_new_folder</code>
